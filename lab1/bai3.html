Programming Langugae( Ngôn ngữ lập trình )

Ngôn ngữ lập trình là một dạng ngôn ngữ máy tính, được dùng để viết ra phần mềm, tập lệnh hoặc các hướng dẫn cho máy tính làm theo.

Scripting Language( Ngôn ngữ kịch bản )

Ngôn ngữ kịch bản đa phần là thông dịch để sử dụng được mà không cần biên dịch. Dễ hiểu thông dịch là viết code tới đâu nó chạy tới đó được, gặp lỗi không nghiêm trọng thì bỏ qua lỗi/ thông báo lỗi rồi chạy tiếp, lỗi nghiêm trọng thì đứng lại đó.

Markup Language( Ngôn ngữ đánh dấu)

Ngôn ngữ đánh dấu Là một ngôn ngữ đơn giản xác định nội dung hiển thị trên màn hình máy tính. Nó sử dụng các thẻ (tag) để xác định các phần tử trong tài liệu (document). HTML và XML là các ngôn ngữ đánh dấu phổ biến nhất.
